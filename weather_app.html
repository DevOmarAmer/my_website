Weather App

Overview

The Weather App is a mobile application that provides real-time weather updates, forecasts, and detailed meteorological data based on the user's location or any city of choice. The app features an intuitive UI for displaying weather conditions, making it easy for users to access crucial weather data at a glance.

Features

  • Real-time weather updates and forecasts
  • Search weather by city or location
  • Detailed weather information (temperature, humidity, wind speed, etc.)
  • Dynamic weather icons based on conditions (sunny, cloudy, rainy, etc.)
  • User-friendly and visually appealing UI
  • Responsive design for both mobile and web platforms

Technologies Used

  • Flutter for cross-platform development
  • Dart programming language
  • Weather API integration for real-time data
  • Firebase for user preferences and data storage
  • Adobe XD for UI/UX design
