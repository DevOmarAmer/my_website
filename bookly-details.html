Bookly App

Overview

The Bookly App is a mobile application designed to help users manage and track their reading habits. The app provides a seamless and intuitive interface for users to add books, track reading progress, and discover new books based on their preferences.

Features

  • Easy book addition and management
  • Reading progress tracking
  • Personalized book recommendations
  • User-friendly and visually appealing UI
  • Responsive design for both mobile and web platforms

Technologies Used

  • Flutter for cross-platform development
  • Dart programming language
  • Firebase for backend services
  • Adobe XD for UI/UX design

Project Gallery

Conclusion

The Bookly App demonstrates expertise in mobile application development and UI/UX design, showcasing a product that is both functional and visually appealing. The project highlights the capability to deliver high-quality applications using Flutter and other modern technologies.
