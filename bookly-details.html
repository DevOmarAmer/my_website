Bookly App

Overview

The Bookly App is a mobile application designed to help users manage and track their reading habits. It offers an intuitive interface for adding books, tracking progress, and discovering new titles based on personal preferences.

Features

  • Book management and tracking
  • Progress tracking by page count
  • Personalized book suggestions
  • User-friendly and visually appealing interface
  • Responsive design for both mobile and web platforms

Technologies Used

  • Flutter for cross-platform development
  • Dart programming language
  • Firebase for backend services
  • Figma for UI/UX design
  • Hive for offline storage
