Fire Detection App

Overview

The Fire Detection App is a mobile solution integrated with an IoT device in vehicles to detect fire or smoke. The app provides real-time alerts through push notifications and sound, allowing users to take immediate action. In addition to alerts, it offers sensor status updates, car location tracking, and detailed fire detection statistics.

Features

  • Real-time fire and smoke detection with instant notifications
  • Push notifications with sound alerts for emergencies
  • Displays sensor status (active/inactive)
  • Tracks car location using GPS
  • Fire detection statistics and history
  • User-friendly interface with responsive design

Technologies Used

  • Flutter for cross-platform development
  • Dart programming language
  • IoT device integration for fire detection
  • Firebase for real-time alerts and data storage
  • GPS for car location tracking
  • Adobe XD for UI/UX design
