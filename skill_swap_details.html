Skill Swap App

Overview

The Skill Swap App is a mobile application designed to connect people who want to exchange skills. The app provides a platform for users to offer their skills and learn new ones from others, promoting a community of shared learning and growth.

Features

  • Profile creation with skill listings
  • Search and match with skill partners
  • Chat functionality for arranging skill swaps
  • Rating and review system for skill exchanges
  • User-friendly and visually appealing UI
  • Responsive design for both mobile and web platforms

Technologies Used

  • Flutter for cross-platform development
  • Dart programming language
  • Firebase for backend services
  • Adobe XD for UI/UX design
